Description

Let's you send a partner an invoice every week/month

How to

1. Go to 'Invoicing->Reoccuring Invoices'
2. Select a partner, this partner will replace the placeholder in the template invoice
3. Select / create a template invoice which will be cloned every month
4. The invoices will be emailed out automaticlly in the background once the next invoice date has past
